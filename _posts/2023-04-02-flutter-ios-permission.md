# 🦥 Info.plist

IOS 의 Permission 을 짧게 정리해보겠다.

애플 뮤직
``` dart
<key>NSAppleMusicUsageDescription</key>
<string>My description about why I need this capability</string>
```
블루투스
``` dart

<string>My description about why I need this capability</string>
```

